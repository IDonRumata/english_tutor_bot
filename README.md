Этот бот — полноценный виртуальный преподаватель, который подстраивается под уровень ученика (от A1 до C1). Система использует мощь LLM (Claude/GPT) для имитации живого общения, глубокого анализа грамматических ошибок и формирования персонального словаря.

---

## 🎯 Бизнес-ценность: Масштабируйте школу, а не ФОТ

Для владельцев онлайн-школ и языковых курсов данный инструмент решает проблему дороговизны человеческих ресурсов:

| Параметр | Живой репетитор | Мой AI-репетитор |
| :--- | :--- | :--- |
| **Стоимость часа** | $20 – $50 | **$0.01 – $0.05** (стоимость токенов) |
| **Доступность** | По расписанию | **24/7**, мгновенный ответ |
| **Масштабируемость** | 1 учитель = 1-15 учеников | **1 бот = ∞ учеников** |
| **Эффективность** | Тратит время на рутину | Берет всю рутину и проверку на себя |

**Идеальный кейс:** Использование бота как "тренажера" между живыми уроками. Ученики практикуются ежедневно, а преподаватель подключается только для сложных тем, что повышает LTV клиента и снижает отток.

---

## 🔥 Ключевые функции

* **Smart Grammar Check:** Глубокая проверка текстов с подробным разбором ошибок на уровне носителя языка.
* **Role-play симуляции:** Свободный диалог на заданные темы (в кафе, собеседование, путешествие) для снятия языкового барьера.
* **Adaptive Vocabulary:** Автоматическое ведение личного словаря ученика на базе слов, вызывающих затруднения.
* **Progress Tracking:** Сохранение истории занятий и наглядная статистика успеваемости.
* **Multi-level Support:** Автоматическая настройка сложности контента под уровни A1-C1.

---

## 🛠 Технический стек

* **Core:** Python 3.11 (Async), aiogram 3.x
* **AI Engine:** Claude API (Anthropic) / OpenAI API (интегрировано через гибкий промпт-инжиниринг)
* **Storage:** SQLite (хранение прогресса, словарей и настроек)
* **Ops:** Docker, Docker-compose, автоматизированный VPS-деплой
